    1. Atributos Globales

    • - ID: Asigna un nombre de identificador único a un elemento, permitiendo su manipulación con CSS o JavaScript.

    • - Class: Asigna uno o más nombres de clase a un elemento para agruparlo con otros y aplicar estilos CSS comunes.

    • - Style: Permite aplicar estilos CSS directamente al elemento.

    • - Lang: Especifica el idioma del contenido dentro de un elemento.

    • - Title: Proporciona texto de ayuda que se muestra al pasar el cursor sobre el elemento.

    • - Contenteditable: Indica si el contenido del elemento es editable por el usuario.

    • - Hidden: Indica que el elemento no debe ser visible para el usuario.

    • - Data-*: Permite almacenar datos personalizados privados para la página o aplicación.

      • 2.Atributos Form
    • - Action: Indica la URL del script o página que procesará los datos del formulario cuando se envíe.
    • - Method: Especifica el método HTTP a utilizar. Los más comunes son get (los datos se adjuntan a la URL) y post (los datos se envían en el cuerpo de la solicitud).
    • - Enctype: Define el modo en que los datos del formulario se codifican al enviarlos. Es especialmente importante para la carga de archivos, donde se suele usar multipart/form-data.
    • - Autocomplete: Activa o desactiva el autocompletado en los campos del formulario. Su valor puede ser on o off.
    • - Accept-charset: Especifica la codificación de caracteres que se utilizará para enviar el formulario.
    • - Novalidate: Un atributo booleano que, si está presente, desactiva la validación HTML5 del formulario antes de su envío.
      • 3. Atributos Input
    • Un input o entrada son los datos o instrucciones que introduces en un programa para que se ejecute. Algunos ejemplos de entrada incluyen escribir en un teclado, hacer clic con el mouse o usar comandos de voz. El input puede ser desde texto simple hasta imágenes complejas o grabaciones de audio.
