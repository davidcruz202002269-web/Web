    1. Qué es C.S.S?

CSS son las siglas de Cascading Style Sheets o Hojas de Estilo en Cascada, un lenguaje utilizado para dar estilo a documentos web escritos en HTML o XML. Separa el contenido (HTML) de su presentación, permitiendo controlar la apariencia de una página web, como los colores, fuentes y disposición de los elementos. CSS es fundamental para el diseño web moderno, ya que facilita la creación de diseños atractivos, responsivos y eficientes.

    2. Cómo utilizar CSS con HTML?

Puedes usar CSS con HTML de tres maneras principales: externo, interno y en línea. El método más recomendado es el externo, donde creas un archivo .css separado y lo vinculas al HTML usando la etiqueta "link". Los métodos interno y en línea se usan para casos más simples, insertando estilos directamente en el HTML con la etiqueta "style" (dentro del "head") o mediante el atributo style en cada etiqueta HTML.

    3. CSS en línea

El CSS en línea es una forma de aplicar estilos de diseño a un único elemento HTML directamente en su etiqueta, utilizando el atributo style. Esto permite definir propiedades como el color, la fuente y el tamaño de manera específica para ese elemento y es útil para tareas puntuales o para asegurar que un estilo se aplique sin ser anulado por otros estilos.

    4. CSS Interno

El CSS interno es código CSS que se define directamente dentro del documento HTML en la sección "head", utilizando la etiqueta "style". Este tipo de CSS se aplica únicamente a la página HTML específica donde se encuentra el código, lo que lo hace ideal para proyectos pequeños o para dar un estilo único a una sola página.

    5. CSS Externo

El CSS externo consiste en escribir las reglas de estilo en un archivo de texto separado con la extensión .css y enlazarlo a los documentos HTML mediante la etiqueta "link" en la sección "head". Es el método más recomendado para proyectos grandes porque permite aplicar estilos a múltiples páginas web a la vez, haciendo que el código HTML sea más limpio, organizado y fácil de mantener al poder modificar el diseño de todo el sitio editando un solo archivo.
