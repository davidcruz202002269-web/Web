    1- Etiquetas de documentos
    • 1.1 HTML:

    • - HTML son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar el contenido de las páginas web. Funciona mediante etiquetas que organizan elementos como textos, imágenes y enlaces, permitiendo que los navegadores web interpreten y muestren la información de manera coherente y accesible.

    • 1.2 Head:

    • - La etiqueta en HTML es una sección que contiene metadatos (datos sobre los datos) de una página web, como el título, enlaces a hojas de estilo, scripts, y otra información que no es visible en la página que el usuario ve en el navegador.

    • 1.3 Body:

    • La etiqueta "base" en HTML sirve para definir una URL base para todos los enlaces y recursos (como imágenes o scripts) de un documento HTML. Esto simplifica la escritura de URLs relativas, ya que no es necesario repetir la dirección completa de la carpeta base en cada enlace. La etiqueta "base" también puede especificar el destino (por ejemplo, en qué ventana o pestaña se abre un enlace) para todas las URL del documento.

      2- Etiquetas de meta datos
    • 2.1 Meta:

    • La etiqueta "meta" en HTML es un elemento que proporciona información sobre el contenido de una página web a los navegadores y motores de búsqueda, sin ser visible para el usuario. Se coloca dentro de la sección "head" del código y contiene metadatos como la descripción, palabras clave, codificación de caracteres y autores. Es fundamental para el SEO y para asegurar la correcta visualización de la página.

    • 2.2 Style:

    • - En HTML, style se refiere tanto a la etiqueta "style" como al atributo style, los cuales se usan para aplicar estilos CSS (Hojas de Estilo en Cascada) a un documento. La etiqueta "style" se coloca dentro de la sección "head" para definir estilos para todo el documento, mientras que el atributo style se aplica directamente a un elemento HTML específico para cambiar su apariencia individual.

    • 2.3 Title:

    • - La etiqueta "title" en HTML define el título de un documento web, que se muestra en la pestaña del navegador, en los resultados de búsqueda de los motores como Google y al guardar la página en favoritos. Este título, también conocido como metatítulo, es fundamental para el SEO, ya que ayuda a los motores de búsqueda a entender el contenido de la página y a los usuarios a decidir si hacer clic en ella.

    • 2.4 Base:

    • La etiqueta "base" en HTML se usa para definir una URL base y un destino de enlace predeterminado para todas las URL relativas en una página, simplificando el código y reduciendo la repetición. Debe colocarse dentro de la etiqueta "head" y solo puede haber una en todo el documento.

    • 2.5 Link:

    • Un "link" en HTML, también conocido como enlace o hipervínculo, es un elemento que permite navegar entre diferentes páginas web, documentos o secciones de una misma página al hacer clic sobre él. Se crea principalmente con la etiqueta "a", utilizando el atributo href para especificar la dirección de destino y el contenido (texto, imagen, etc.) entre las etiquetas de apertura y cierre.

      3- Etiquetas de Agrupación
    • 3.1 p:

    • En HTML, "p" es una etiqueta que se usa para definir un párrafo de texto. Cuando se coloca texto entre las etiquetas de apertura "p" y de cierre "/p", se indica al navegador que ese contenido debe ser tratado como un párrafo, creando un bloque de texto separado del contenido adyacente por un espacio en blanco.

    • 3.2 hr:

    • significa "horizontal rule" o "línea horizontal" y se usa para crear una separación temática entre secciones de contenido. Se muestra como una línea horizontal que divide visualmente el texto, facilitando la organización y legibilidad de una página web.

    • 3.3 ul:

    • - Es la etiqueta que define una lista desordenada, es decir, una lista donde los elementos no tienen un orden específico y se presentan típicamente con viñetas. La etiqueta "ul" debe contener una o más etiquetas "li" (list item) para cada uno de los elementos que forman la lista.

    • 3.4 div:

    • - Es una etiqueta de HTML que se utiliza como un contenedor genérico para agrupar otros elementos, permitiendo dividir el contenido de una página en secciones lógicas. No tiene un significado semántico propio, por lo que su función principal es organizar el código para que sea más fácil dar estilo a bloques de contenido mediante CSS.

    • 3.5 figure:

    • - La etiqueta "figure" en HTML se usa para agrupar contenido independiente como imágenes, videos o ilustraciones, que puede ser movido del flujo principal del documento sin afectar su significado. Se utiliza junto con la etiqueta "figcaption" para proporcionar un título o descripción a ese contenido.

    • 3.6 figcaption:

    • - La etiqueta "figcaption" en HTML se usa para proporcionar una leyenda o pie de foto a un elemento multimedia, como una imagen, un video o un gráfico, que se encuentra dentro de la etiqueta contenedora "figure". Su propósito principal es mejorar la accesibilidad y la estructura del contenido, ofreciendo un título o comentario visible en pantalla que contextualiza el elemento al que está asociado.

      4- Etiquetas Textuales
    • 4.1 a:

    • - La etiqueta "a" en HTML es la etiqueta de ancla, utilizada para crear hipervínculos que enlazan a otras páginas web, archivos, ubicaciones dentro de la misma página o direcciones de correo electrónico. Se utiliza junto con el atributo href para especificar la URL de destino y con otros atributos, como target, para controlar el comportamiento del enlace, por ejemplo, si se abre en una nueva pestaña.

    • 4.2 span:

    • - La etiqueta en HTML es un contenedor genérico en línea que se utiliza para agrupar elementos o aplicar estilos a una sección de texto sin afectar el flujo del documento. Por sí sola no tiene ningún efecto visual, pero se usa comúnmente con CSS para dar formato a partes específicas de un texto o con JavaScript para manipular segmentos del documento.

    • 4.3 sub:

    • - La etiqueta "sub" en HTML se utiliza para representar texto en subíndice, es decir, texto que aparece medio carácter por debajo de la línea base y, a menudo, con una fuente más pequeña. Es útil para elementos como fórmulas químicas (por ejemplo, $H_2O$), ecuaciones matemáticas o notas al pie de página.

    • 4.4 sup:

    • - La etiqueta en HTML sirve para marcar texto que se debe mostrar como superíndice, es decir, elevado por encima de la línea normal del texto. Se utiliza para propósitos semánticos, como para exponentes en matemáticas (\(x^{2}\)) o notas al pie, no para estilos de presentación tipográfica por sí solos

    • 4.5 cite:

    • - La etiqueta "cite" en HTML representa el título de una obra creativa (como un libro o una película) o una referencia a un autor, y por defecto se muestra en cursiva. Es un elemento semántico que indica que el texto dentro de ella es una cita, lo cual ayuda a los motores de búsqueda a comprender el contenido, aunque para citas cortas en línea se recomienda usar la etiqueta "q".

      5- Etiquetas Semanticas
    • 5.1 nav:

    • La etiqueta "nav" en HTML representa una sección de una página web que contiene enlaces de navegación a otras páginas o a partes dentro de la misma página. Su propósito principal es estructurar el contenido de navegación para que sea más accesible para los usuarios y para los motores de búsqueda, mejorando la semántica y la organización del sitio web.

    • 5.2 section:

    • - La etiqueta

    en HTML se utiliza para agrupar contenido temáticamente relacionado, creando secciones distintas y coherentes dentro de una página web

    • 5.3 article:

    • - La etiqueta

    en HTML representa un contenido autónomo y autocontenido que puede distribuirse o reutilizarse independientemente del resto del documento.

    • 5.4 header:

    • - El header en HTML es una etiqueta semántica de HTML5 que define la sección de encabezado de una página web o de una sección dentro de ella. Su propósito es agrupar contenido introductorio, como el logotipo, el título principal, el menú de navegación y otros elementos relacionados. Se ubica en la parte superior de la página y usualmente contiene elementos esenciales que se repiten en varias páginas.

    • 5.5 main:

    • - La etiqueta "main" en HTML se utiliza para definir el contenido principal de un documento, el cual debe ser único en esa página y no repetirse en otras, como menús de navegación o pies de página. Su propósito es semántico y de accesibilidad, ya que ayuda a navegadores, motores de búsqueda y lectores de pantalla a identificar el contenido más relevante de la página

    • 5.6 footer:

    • Un "footer" en HTML es una sección de pie de página ubicada en la parte inferior de una página web o de una sección específica de contenido, que generalmente contiene información como los derechos de autor, información de contacto, enlaces legales y un mapa del sitio. Su propósito es organizar y estructurar el contenido, aunque no sea el principal, proporcionando detalles importantes y una navegación adicional.
